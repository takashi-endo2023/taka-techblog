> このテーブルが収益源ステータスの **正（single source of truth）**。
> 他ドキュメントはここを参照する。

### 短期計画（3ヶ月以内）

#### 既存アフィリエイトの最適化
- レバテック2件の承認待ち → 承認後に AWS/DevOps 記事に配置
- A8 で他のIT系プログラム探索（Findy、フリーランススタート等）

#### サーバー・ツール紹介
- ConoHa / Xserver / さくらインターネット の紹介プログラム
- インフラ記事の文脈で自然に配置可能
- 単価は高めだが成約難度も高い

#### Udemy アフィリエイト（impact.com 経由）✅登録済
- **配置戦略の詳細は `affiliate-strategy.md` の Udemy セクション**
- 「独学派」ブランドと一致。career-change（未経験）にも置ける＝GEEKLY が届かない層をカバー
- 報酬は **USD建て**（円選択不可・仕様）。為替の影響は金額が小さいうちは誤差
- **受取/税務の段取り（初回支払い前までに）**:
  - Finance → Bank Account（受取方法・Payoneer 等の日本対応を選択）
  - Finance → Tax Information で **W-8BEN** 提出（出さないと報酬30%源泉徴収。日米租税条約で軽減）
  - 最低支払額に達するまで出金不可 → 当面は貯まるのを待つ。今は設定不要
- セール時（1,200円〜）を狙う旨を添えると CV が上がる

### 中期計画（3〜6ヶ月）

#### note 有料記事 → 詳細は `note-strategy.md`
- 「ブログでは書けない深い話」を有料化
- 例: 「GEEKLY との実体験」「年収交渉の具体的やりとり」
- 単価500〜2000円、月数十部で固定収入化
- ブログでフックを作り note に誘導
- **第1弾候補: 「同僚が部下になった日」完全版（500円）**

#### 寄稿・記事執筆代行
- 技術メディアからの依頼を受ける
- Findy Engineer Lab、レバテックブログ、Qiita Zine 等
- 1記事数万円〜、自分の知名度も上がる

#### 講演・登壇（優先度低・将来の選択肢）
- ハードルが高いため当面は追わない。Contact では「執筆・寄稿」のみ受付
- やるとしても福岡のローカルイベントから。実績とフォロワーが育ってから検討
- 数千〜数万円 + ネットワーク拡大

### 長期計画（6ヶ月〜1年）

#### オンライン講座
- Udemy 講座出版（Claude Code 活用、テックリード入門等）
- 単価1500〜6000円、買い切り
- 一度作れば長期的な収益源

#### 書籍出版
- 商業出版（編集者に拾われる）
- ブログ実績 + X フォロワーが必要（フォロワー1000+目安）
- 印税は10〜15%、初版数千部で数十万円

#### コンサルティング
- 企業向け技術支援
- Claude Code 導入、AI開発内製化、テックリード育成
- 月数万〜数十万円、副業 or 独立準備

#### Patreon / メンバーシップ
- 月額有料コミュニティ
- フォロワー数千人規模になってから
- 月500〜2000円 × 数十人で月数万円

### マネタイズの判断基準

#### やる
- ブログ・X のコンテンツと一貫性がある
- 読者の利益になる（騙すような誘導はしない）
- 規約・法律に準拠
- 自分の時間効率に見合う単価

#### やらない
- ブランドを毀損する案件
- 自分が使ったことのない商品の宣伝
- ステマ・誇大広告
- 競合との二股（GEEKLY 推しながら他社推し等）

## 4. ブランド軸の維持

成長過程で「ブレない軸」を保つことが長期的な収益化のカギ。

### コア軸
- **AI / Claude Code 実務活用**
- **テックリード / 組織作り**
- **医療IT / 規制産業の現実**
- **30歳転身 / キャリアチェンジ**

### 軸から外さないこと
- 副業案件・転職エージェント案件は **「自分で使った/使う想定」のみ**
- 講座・書籍も **自分の専門領域のみ**
- 講演テーマも軸から外れないこと
