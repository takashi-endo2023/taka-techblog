## 1. 基本思想

### 媒体ごとの役割分担

| 媒体 | 役割 | 課金 |
|---|---|---|
| ブログ | SEO・認知・AdSense・アフィリ | 無料 |
| Zenn | 技術認知・被リンク | 無料 |
| X | 拡散・フック | 無料 |
| **note** | **ブログで書けない深い実体験・ノウハウ** | **有料** |

### 鉄則

> ブログに書いてる内容を note で有料化しても誰も買わない。
> note は「ここでしか読めない生々しさ・具体性」を売る。

| ブログ（無料・きれいな教訓） | note（有料・生々しい詳細） |
|---|---|
| 内製化で480万円削減 | 経営層をどう説得したか、反対意見、年収交渉の中身 |
| テックリード就任90日 | 同僚が部下になった気まずさ、給与差、人間関係 |
| AIで本部長がモンスターに | その後の収拾、社内政治の全記録 |

## 3. 価格戦略

| カテゴリ | 価格 | 理由 |
|---|---|---|
| 入門・体験談 | 500円 | 買いやすさ重視。初購入の心理ハードルを下げる |
| ノウハウ・実務 | 980〜1500円 | 中身の濃さで正当化 |
| 超ニッチ高単価（医療IT規制） | 2000〜3000円 | 競合不在、必要な人には安い |

### 価格テクニック

- **早期割引**: 「最初の10部は500円、以降980円」で初動を作る
- **返金保証**: note の返金機能を明記して購入ハードルを下げる
- **冒頭無料**: クライマックス手前に有料ラインを設定し「続きが気になる」状態で課金

## 4. 投稿フロー（ブログ連携）

```
① ブログで無料記事公開（フック・SEO）
   例: 「内製化した3年間」きれいな教訓版
        ↓
② X でブログ告知 + 「詳しい裏側は note に」と予告
        ↓
③ note で有料版公開（生々しい詳細・数字・会話）
   例: 「経営層を説得した実際のやりとり全文」
        ↓
④ ブログ記事末尾に note リンク設置
   「この経緯の詳細は note に書きました（有料）」
```

→ 無料記事が有料記事の広告になる構造。

## 5. 運用スパン

- 質を最優先、実績ゼロから

- マガジン化を検討

- 定期購読マガジン立ち上げ
- 月額500〜1000円 × 購読者

## 7. note 記事の構成テンプレ

```
【タイトル】具体的・生々しさを匂わせる
　例: 「同僚が部下になった日——給与差と気まずさの3ヶ月」

【冒頭（無料）】
　- 一番引き込まれるシーン
　- 「この続きが読みたい」と思わせる
　- 1000〜1500字程度

──────── ここから有料 ────────

【本編（有料）】
　- 無料では書けない具体・数字・会話
　- 失敗の詳細
　- 心情の生々しさ

【締め】
　- 読者が持ち帰れる学び
　- 他の note / ブログへの導線
```

## 8. リスク・注意点

### ⚠️ 守秘義務・コンプライアンス
医療IT・社内政治の話は特定されない加工が必須。
- 会社名・製品名・人名は完全に伏せる
- 「本部長」等の役職もぼかす
- 治験データの具体は絶対に出さない

### ⚠️ ブランド毀損リスク
「金儲けに走った」と見られないバランス。
無料コンテンツの質を保ちながら有料を出す。

### ⚠️ 時間対効果
note 1本に数時間かける価値があるか。
最初は「ブログ記事の深掘り再利用」で工数を抑える。

### ⚠️ Zenn からの導線
Zenn 規約上アフィリリンクは NG。note リンクの可否は要確認。
基本は X・ブログから note へ誘導する。
